# 🍎Yeolmae

![main_logo](https://github.com/CapstoneBase/Yeolmae-front/assets/118517380/cac2e33a-46c4-442a-b2b7-c45f99745c14)



## 프로젝트 설명
졸업 작품 공유 및 아카이빙 서비스

## 주요 기능
- 전공 분류 별 졸업 작품을 열람할 수 있습니다.
- 댓글을 통해 네트워킹이 가능합니다.
- 프로필을 통해 작품 제작자의 연락처를 확인할 수 있습니다.

## 프로젝트 기간
2024.03.10 ~ 2024.06.13  (이후 계속 진행 예정)
